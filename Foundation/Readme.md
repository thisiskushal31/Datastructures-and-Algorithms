### Foundation

**Foundation** contains fundamental concepts and building blocks for learning Data Structures and Algorithms.

This folder is designed to help beginners build a strong foundation before diving into advanced topics.

#### Contents

| File | Description |
|------|-------------|
| [Coding_Patterns.md](./Coding_Patterns.md) | 16 Essential coding patterns for solving DSA problems during interviews |
| [Pattern_Identification_Cheatsheet.md](./Pattern_Identification_Cheatsheet.md) | Keyword-based pattern identification guide for quick problem recognition |

#### Key Topics

- **Coding Patterns** - 16 essential patterns for problem-solving
  - Sliding Window
  - Two Pointers
  - Fast & Slow Pointers
  - Merge Intervals
  - Cyclic Sort
  - In-place Reversal
  - Tree BFS/DFS
  - Two Heaps
  - Subsets
  - Modified Binary Search
  - Bitwise XOR
  - Top K Elements
  - K-way Merge
  - 0/1 Knapsack (DP)
  - Topological Sort

#### Recommended Learning Path

1. Start with **Logic Building** in [Algorithms/00_Logic_Building.md](../Algorithms/00_Logic_Building.md)
2. Learn **Pattern Identification** using [Pattern_Identification_Cheatsheet.md](./Pattern_Identification_Cheatsheet.md) - Quick keyword-based pattern recognition
3. Master **Coding Patterns** in [Coding_Patterns.md](./Coding_Patterns.md) - Detailed pattern explanations
4. Learn **Time Complexity and Space Complexity** in [Algorithms/01_Time_Complexity_and_Space_Complexity.md](../Algorithms/01_Time_Complexity_and_Space_Complexity.md)
5. Practice basic **Searching and Sorting** in [Algorithms/02_Searching_and_Sorting.md](../Algorithms/02_Searching_and_Sorting.md)
6. Explore **Data Structures** in [DataStructures/Readme.md](../DataStructures/Readme.md)
7. Master **Algorithms** in [Algorithms/Readme.md](../Algorithms/Readme.md)
